# Animal Voice Translator — 动物叫声翻译产品

用 AI 解读宠物情绪，实现跨物种"交流"。

## 项目概览

产品定位：基于 AI 大语言模型 + 动物声学分析的宠物叫声翻译工具。  
目标用户：养宠人群（猫/狗为主，后续扩展）。  
核心理念：**透明翻译** — 不仅给结论，更展示推理过程和置信度。
